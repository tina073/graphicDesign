(1) 안내문이나 경고 등의 간단한 메시지를 호출하게 해주는 [alert( ) 함수]
(2) 확인과 취소를 할 수 있는 [confirm( ) 함수]
(3) 입력 대화상자를 만들어주는 [prompt( ) 함수]
(4) 문자열을 자바스크립트 구문으로 바꿔주는 [eval( ) 함수]
(5) 문자열을 실수로 바꿔주는 [parseFloat( ) 함수]
(6) 문자열을 정수로 바꿔주는 [parseInt( ) 함수]
(7) 문자셋과 아스키(ASCII) 값을 반환해주는 [escape( )]와 [unescape( ) 함수]
(8) 숫자인지 여부를 알려주는 [isNaN( ) 함수]
(9) 문자로 변환하여 반환하는 [string() 함수]
(10) 숫자로 변환하여 반환하는 [Number() 함수]
