현재 첫번째 박스의 위쪽공간이 20픽셀인 상황!!!

멀티클래스는 여러개의 클래스네임을 같이 사용하는 것을 말한다.
안쪽박스의 클래스 네임을 전부 [one]으로 주고 위쪽으로 마진을 10픽셀을 주었다면
안쪽 첫번째 박스의 마진이 wrapper에 준 패딩이랑 겹쳐서 위쪽이 총 20픽셀의 공간이 생긴다.

이때 첫번째 박스에 한칸 띄어 클래스 네임을 더 주고
.one.two 식으로 사용하여 마진을 0으로 만들면 매 위쪽에 들어간 마진 10픽셀이 빠진다.

스타일이 중복이 될 경우 클래스 네임 한개보다 클래스 네임 두개를 쓴것이 우선되므로
첫번째 .one에서 안쪽박스들의 마진을 위쪽으로 전부 준 후에
두번째 .one.two에서 첫번째 박스의 위쪽으로 준 마진 하나가 다시 빠진다.
